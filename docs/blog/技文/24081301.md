### 1.下载插件

```bash
npm i vuepress-plugin-side-anchor -D
```

### 2.使用

在 `.vuepress/config.js` 添加以下配置

```javascript
module.exports = {
  plugins: [
    // ...
    ["vuepress-plugin-right-anchor"],
  ],
};
```

### 3.样式更改

在 `.vuepress/styles/index.styl` 更改样式

```css
.side-anchor{
  border-left: 0.1rem #858585 solid
}
.side-anchor-link{
  color  var(--text-color)
}
.side-anchor-link.active{
  color  #3eaf7c
}
```
