<Question :questions="['在 Vue 渲染模板时，如何保留模板中的 HTML 注释']" />

---

在 Vue 中，Vue 模板编译时会默认移除 HTML 注释，这是出于性能考虑，减少不必要的 DOM 节点。
```html
<template>
  <!-- 这个注释会被移除 -->
  <div>内容</div>
</template>

<!-- 渲染结果 -->
<div>内容</div>
```

## comments 选项

Vue2 中，可以通过设置 `comments` 选项为 `true` 来保留模板中的 HTML 注释。

::: demo vue"
```vue
<script>
export default {
  data() {
    return {
      count: 10
    }
  },
  comments: true
}

</script>

<template>
  <div class="counter">
    <!-- 显示计数器 -->
     <p>计数：{{ count }}</p>
  </div>
</template>
```
:::
