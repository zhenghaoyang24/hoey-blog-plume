<Question :questions="['CSS 中哪些属性会继承？']" />

---

CSS 中有些属性能够继承，会从父元素传递给子元素。不可继承的属性则只应用于定义它的元素。可继承属性主要涉及字体及文本相关的属性。

## 可继承属性

color、font-family、font-size、font-style、font-variant、font-weight、letter-spacing、line-height、
text-align、text-indent、text-transform、visibility、white-space、word-spacing

## 不可继承属性

background、border、display、margin、padding、position、z-index、float、clear

## 默认值和继承

CSS 提供三个属性值用于空值属性的继承性：inherit、initial、unset。

- inherit：表示继承父元素的属性值。
- initial：表示使用属性的默认值。
- unset：对于可继承性，行为和 inherit 一样；对于不可继承性，行为和 initial 一样。
