## 新的语义化标签

增加了语义化的标签，如 `<header>`、`<footer>`、`<nav>`、`<section>`、`<article>`、`<aside>` 等，
有利于 SEO 和 无障碍访问。

## 文档类型声明

HTML5 引入了 `<!DOCTYPE html>` 文档类型声明，取代了以往较复杂的文档声明。

## 多媒体支持

通过 `<audio>` 和 `<video>` 标签支持原生视频和音频，不再依赖 Flash 插件。

## 表单控件增强

HTML5 引入了更多的表单类型，如 `<input type="email">`、`<input type="number">`、`<input type="color">` 等，
引入了新属性 `placeholder`、`autofocus`、`autocomplete` 等，提高了表单的数据校验能力。

## 本地存储

HTML5 引入了 `localStorage` 和 `sessionStorage`，提供了客户端存储数据能力。

## 绘图与图形

引入 `<canvas>` 标签，支持通过 JavaScript 绘制 2D 图形，增强对 SVG 的支持。
