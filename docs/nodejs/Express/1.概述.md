`Express.js` 是一个开源的 Web 应用程序框架，基于 Node.js 平台构建，是最著名的 Node.js 后端框架之一。它强大而灵活，轻量且易于学习，许多后端框架都基于 `Express.js` 开发。
`Express.js` 可用于中小型项目，还是构建个人 API 的最佳选择之一。

下文不是 `Express.js` 的详细教程，只是记录一些用法与特性，便于复习。
